# Data Structures Implementation (C++)

A collection of custom data structure implementations built from scratch in C++ without relying on STL containers. The project demonstrates the internal design and implementation of fundamental data structures using templates, pointers, dynamic memory allocation, and object-oriented programming principles.

### Implemented Data Structures

- Dynamic Array
- Doubly Linked List
- Queue (Linked List Implementation)
- Stack (Linked List Implementation)
- Queue (Dynamic Array Implementation)
- Stack (Dynamic Array Implementation)

### Features

- Generic implementations using C++ templates
- Dynamic memory management
- Insert, delete, update, search, and traversal operations
- Reusable and extensible design
- Multiple implementations for the same data structure
- Queue and Stack implementations built on top of custom containers

### Concepts Practiced

- Data Structures
- Templates
- Pointers
- Dynamic Memory Allocation
- Object-Oriented Programming (OOP)
- Inheritance
- Composition
- Generic Programming
- Memory Management

### Technologies

- C++
- Header-Only Classes
- Console-Based Testing

## Purpose

This project was developed to gain a deeper understanding of how fundamental data structures work internally by implementing them from scratch rather than relying on the Standard Template Library (STL).
